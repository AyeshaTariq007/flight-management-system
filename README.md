# Flight Management System

A comprehensive relational database system designed to manage and streamline airline operations, passenger bookings, and flight scheduling.

## Features
- Architected optimized relational database schemas for tracking airline matrices and customer records.
- Implemented multi-table storage constraints, primary/foreign keys, and normalized structural joins.
- Designed high-performance SQL queries for automated data extraction and scheduling updates.
- Ensured data integrity and prevention of redundancy through proper database normalization.

## Tech Stack
- Database Engine: MySQL / Microsoft SQL Server
- Language: SQL (Structured Query Language)
